Classpublic class UnitOfWork

The UnitOfWork is a standard enterprise design pattern used to delay writes to the database. In some senses a UnitOfWork is similar to a database transaction (and in fact the UnitOfWork is always executed as a transaction on the server). This design pattern is particularly relevant to an asynchronous client/server library like Flextrine.
